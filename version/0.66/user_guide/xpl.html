File explorer overview#

A specific property under the Mapdl class is dedicated for XPL. It’s based on the APDL Math *XPL command and can be used to explore binary files generated from APDL.

All APDL files are organized the same way, as a tree or records. For example, a binary result file has the structure outlined in the Structure of an APDL Result figure.

You can find a detailed example of the APDL file explorer in Binary MAPDL File Explorer.

For more information, see Database explorer.
