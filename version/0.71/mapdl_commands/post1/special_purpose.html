class ansys.mapdl.core._commands.post1.special_purpose.SpecialPurpose#

SpecialPurpose.bfint([fname1, ext1, fname2, ...])

Activates the body force interpolation operation.

SpecialPurpose.cbdof([fname1, ext1, fname2, ...])

Activates cut-boundary interpolation (for submodeling).

SpecialPurpose.cmsfile([option, fname, ext, ...])

Specifies a list of component mode synthesis (CMS) results files for plotting results on the assembly.

SpecialPurpose.cyccalc([fileprefix, ...])

Calculates results from a cyclic harmonic mode-superposition analysis using the specifications defined by cycspec.

SpecialPurpose.cycfiles([fnamerst, extrst, ...])

Specifies the data files where results are to be found for a cyclic symmetry mode-superposition harmonic analysis.

SpecialPurpose.cycphase([type_, option])

Provides tools for determining minimum and maximum possible result values from frequency couplets produced in a modal cyclic symmetry analysis.

SpecialPurpose.cycspec([label, node, item, comp])

Defines the set of result items for a subsequent cyccalc command in postprocessing a cyclic harmonic mode-superposition analysis.

SpecialPurpose.exoption([ldtype, option, value])

Specifies the exprofile options for the Mechanical APDL to Ansys CFX profile file transfer.

SpecialPurpose.expand([nrepeat, modal, ...])

Displays the results of a modal cyclic symmetry analysis.

SpecialPurpose.exprofile([ldtype, load, ...])

Exports Mechanical APDL interface data on selected nodes to an Ansys CFX Profile file.

SpecialPurpose.exunit([ldtype, load, ...])

Specifies the interface data unit labels to be written to the profile file from Mechanical APDL to Ansys CFX transfer.

SpecialPurpose.fssparm([port1, port2])

Calculates reflection and transmission properties of a frequency selective surface.

SpecialPurpose.fsum([lab, item])

Sums the nodal force and moment contributions of elements.

SpecialPurpose.hfang([lab, phi1, phi2, ...])

Defines or displays spatial angles of a spherical radiation surface for sound radiation parameter calculations.

SpecialPurpose.hfsym([kcn, xkey, ykey, zkey])

Indicates the presence of symmetry planes for the computation of acoustic fields in the near and far field domains (beyond the finite element region).

SpecialPurpose.macopt(**kwargs)

Specifies modal assurance criterion (MAC) or frequency response function (FRF) correlation criteria calculation options for rstmac.

SpecialPurpose.nforce([item])

Sums the nodal forces and moments of elements attached to nodes.

SpecialPurpose.nldpost([label, key, fileid, ...])

Gets element component information from nonlinear diagnostic files.

SpecialPurpose.plas([lab, ldstep, substep, ...])

Plots a specified acoustic quantity during postprocessing of an acoustic analysis.

SpecialPurpose.plcamp([option, slope, unit, ...])

Plots Campbell diagram data for applications involving rotating structure dynamics.

SpecialPurpose.plcfreq([spec, sectbeg, sectend])

Plots the frequency response for the given cycspec specification.

SpecialPurpose.plchist([spec, freqpt])

Plots a histogram of the frequency response of each sector for the given cycspec specification.

SpecialPurpose.plfar([lab, option, var1b, ...])

Plots pressure far fields and far-field parameters.

SpecialPurpose.plmc([lstep, sbstep, timfrq, ...])

Plots the modal coordinates from a mode-superposition solution.

SpecialPurpose.plnear([lab, opt, kcn, val1, ...])

Plots the pressure in the near zone exterior to the equivalent source surface.

SpecialPurpose.plzz([rotvel, deltarotvel])

Plots the interference diagram from a cyclic modal analysis.

SpecialPurpose.pras([lab, ldstep, substep, ...])

Prints a specified acoustic quantity during postprocessing of an acoustic analysis.

SpecialPurpose.prcamp([option, slope, unit, ...])

Prints Campbell diagram data for applications involving rotating structure dynamics.

SpecialPurpose.prfar([lab, option, var1b, ...])

Prints acoustic far field parameters.

SpecialPurpose.prmc([lstep, sbstep, timfrq, ...])

Prints the modal coordinates from a mode-superposition solution.

SpecialPurpose.prnear([lab, opt, kcn, val1, ...])

Prints the pressure in the near zone exterior to the equivalent source surface.

SpecialPurpose.reswrite([fname, cflag])

SpecialPurpose.rmflvec(**kwargs)

SpecialPurpose.rsplit([option, label, ...])

SpecialPurpose.rstmac([file1, lstep1, ...])

SpecialPurpose.slashexpand([nrepeat1, ...])

SpecialPurpose.spmwrite([method, nmode, ...])

SpecialPurpose.spoint([node, x, y, z, ...])
