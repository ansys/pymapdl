Connection commands#

These commands read in external CAD files into MAPDL.

Mapdl.cat5in([name, extension, path, ...])

Transfers a .CATPart file into the ANSYS program.

Mapdl.catiain([name, extension, path, blank])

Transfers a CATIA model into the ANSYS program.

Mapdl.parain([name, extension, path, ...])

Transfers a Parasolid file into the ANSYS program.

Mapdl.proein([name, extension, path, proecomm])

APDL Command: ~PROEIN

Mapdl.satin([name, extension, path, entity, ...])

Transfers a .SAT file into the ANSYS program.

Mapdl.ugin([name, extension, path, entity, ...])

Transfers an NX part into the ANSYS program.
